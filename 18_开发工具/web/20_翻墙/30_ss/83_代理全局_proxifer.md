

# 概述


    Proxifier是一款功能非常强大的socks5客户端，
    可以让不支持通过代理服务器工作的网络程序能通过HTTPS或SOCKS代理或代理链。
    但是速度会慢很多,建议可以直接使用软件提供的代理设置

# 使用

设置代理

    profile --> proxy server --> 设置为ss客户端
    
    点击test,测试代理

    
设置规则

    profile --> profication rule  
    
    设置规则,并启用规则就好
    请求匹配任何一个规则都会执行
    default规则是启用的(让他直连就好),

查看
    
    在首页就能看到你各个软件的连接情况
